# PrimeTech Segurança & Elétrica

Site institucional da **PrimeTech Segurança & Elétrica**, desenvolvido em React + Vite.

## Sobre o projeto

Landing page profissional para divulgação de serviços de segurança eletrônica, elétrica, redes e infraestrutura técnica em Penedo e região.

## Serviços apresentados

- Instalação de câmeras de segurança
- Sistemas de alarme
- Cerca elétrica e concertina
- Redes e internet
- Cabeamento estruturado
- Manutenção e suporte técnico

## Tecnologias utilizadas

- React
- Vite
- CSS
- Lucide React
- GitHub Pages
